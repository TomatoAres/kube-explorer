# kube-explorer

本书旨在系统讲解 Kubernetes 核心组件原理，帮助读者快速入门 Kubernetes!

## kubernetes 简述

kubernetes 简称 k8s，是一个容器调度工具，它提供了强大的部署、运维、扩缩容能力。
